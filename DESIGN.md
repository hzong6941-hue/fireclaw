# DESIGN.md — 火牛AIGC 品牌官网

## 气质与意象

深夜太空舱，数字火焰在虚空中升腾。一头火牛从像素裂隙中踏出，牛角化为数据洪流。不是传统企业官网，而是一座沉浸式品牌数字展厅——像劳斯莱斯的线上艺术馆，又像 Apple 发布会的叙事节奏。克制、深邃、只在关键时刻释放能量。IP吉祥物如同幽灵般穿梭在页面各处，时而远虚时而近浮，是整个数字展厅的"灵"。

## 视觉策略

- **暖色深色沉浸**：背景以 #0A0A14 为最深色，叠加多层 radial-gradient 暖色光晕（橙/紫/琥珀），营造温暖而深邃的空间感
- **IP分层背景**：吉祥物以4层不同尺寸/虚化/透明度/动画分布在背景中，远虚近实，模拟景深
- **渐变克制**：只在关键视觉焦点使用橙→明橙渐变，不滥用紫色
- **毛玻璃卡片**：rgba(255,255,255,0.03) + backdrop-blur，浮在深色底上
- **全屏叙事**：每个区块占满视口，滚动即切换"展厅"
- **火焰粒子**：Hero区背景用Canvas粒子，模拟数字火焰，升级至80个粒子+6种色彩
- **IP吉祥物融入**：在每个section以不同形式出现——浮层背景、角落水印、徽章、浮动动画

## 字体排版

- 中文标题：思源黑体 Noto Sans SC (Google Fonts CN)，weight 700/900
- 英文标题：Inter，weight 600/800
- 正文：思源黑体 weight 400，行高 1.8
- 标题字号节奏：Hero 72px → 区块标题 48px → 卡片标题 24px → 正文 16px
- 大写英文用于标签/分类词，字间距 0.15em

## 动效与交互

- 滚动驱动：IntersectionObserver 触发淡入上浮 (translateY 50px → 0, opacity 0 → 1)
- 缓动曲线：cubic-bezier(0.16, 1, 0.3, 1) — 快出慢停，高端感
- 悬停反馈：卡片悬停边框从透明渐变为炽焰橙，scale 1.01
- 导航栏：滚动后毛玻璃固定，z-50
- 数字增长：关键数字滚动增长动画
- IP吉祥物动画：float(6s)、floatSlow(8s)、breath(5s)、drift(20s) 四种动画
- 案例卡片：Banner渐变+品牌首字母+悬停缩放

## 页面结构

1. **Hero** — 暖色径向渐变背景 + 4层IP浮层 + 火焰粒子 + Logo+IP组合 + 品牌Slogan
2. **品牌Marquee** — 滚动品牌名 + IP远景装饰
3. **关于我们** — 暖色氛围光 + IP浮层 + 卡片角落IP水印
4. **核心优势** — 渐变背景 + IP左侧装饰 + 卡片角落IP
5. **AIGC服务** — 多层氛围光 + IP双浮层 + 标题旁IP徽章
6. **案例展示** — 升级版画廊卡片(视觉Banner+品牌首字母+核心成果) + 分类筛选 + 详情浮层(带Banner+成果展示)
7. **数据指标** — 暖色光晕 + IP右上浮层 + 卡片角落IP
8. **全链路服务** — 紫色氛围光 + IP左侧浮层 + 流程步骤
9. **创始人战绩** — 暖色氛围 + 卡片角落IP
10. **品牌基因** — Logo+IP双卡片(带氛围光+浮动动画) + 品牌色系
11. **联系/CTA** — 暖色大光晕 + 双层IP浮层 + 浮动IP + 联系方式
12. **Footer** — 品牌色条 + Logo+IP + 链接

## 案例卡片视觉规范

- 顶部视觉Banner：高度144px，品牌专属渐变色
- Banner内大品牌首字母：6xl-7xl font-black，白色20%透明度
- Banner内IP水印：40px，10%透明度
- Banner底部渐变遮罩，融入卡片主体
- 卡片内核心成果：2条，小圆点+品牌色，truncate
- 详情浮层：顶部同款Banner + 项目成果专区

## 设计禁忌

- 禁止使用科技蓝/赛博紫作为主色调（品牌色是炽焰橙）
- 禁止白色/浅色背景大面积使用
- 禁止卡通/可爱风格（尽管IP是chibi风，但网页整体是高端克制）
- 禁止超过3种字重混用
- 禁止全大段纯文字，每段≤80字
- 禁止IP吉祥物遮挡文字内容，透明度不得高于0.08（近景最高0.08）
